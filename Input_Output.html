1.

Crie um programa que mostre numa caixa de diálogo “Alert” a mensagem: “É preciso fazer todos os exercícios para aprender”.
A caixa “Alert” será exibida depois do utilizador clicar no botão.

2.

Crie um programa que mostre uma caixa de diálogo “Confirm” a mensagem: “Deseja continuar a realizar os exercícios?”
a. Se o utilizador clicar “Ok” mostre a mensagem “Bem escolhido”
b. Se o utilizador clicar “Cancelar” mostre a mensagem “Já chega de JavaScript”

3.

Crie um programa que leia o nome do utilizador através de uma caixa de diálogo “Prompt” e mostre na página Web a mensagem: “O(A) Formando(a)” + [nome formando] + ” está aprovado”. Se o utilizador clicar em “Cancelar” não deverá ser exibido a mensagem.

4.

Crie um programa que leia o nome e idade do utilizador através de caixas de texto e mostre na página Web a mensagem: “O(A) Formando(a) ” + [nome formando] + ” tem ” + [idade formando ] + “ anos”.

5.

Crie um programa que mostre na “console” a mensagem: “Eu sou o “ [nome formando] “ e sou adepto do ” [clube de futebol].

6.

Crie um programa que leia do utilizador os seguintes dados e de seguida mostre-os num parágrafo:
• Nome: [Nome do formando]
• Estado civil: (Casado; Solteiro; Divorciado; Viúvo)
• Idade: [Idade formando]
• Altura: [Altura do formando]
• Sócio: [Sim/Não]
• Email: [Email do formando]

7.

Crie um programa que mostre na página Web a data atual do sistema.

8.

Crie um programa que mostre na página Web a mensagem “Bom dia”, “Boa tarde”, “Boa noite” em conformidade com a hora do sistema.

9.

Crie um programa com um botão que permita imprimir (na impressora) a página Web.

10.

Crie um programa com um botão que mostre o endereço (URL) da página Web.
